# Microsoft 365 Security Engineering

This repository contains practical notes, workflows, and examples based on hands-on experience securing Microsoft 365 environments.

## Focus Areas
- Identity security (Entra ID / Azure AD)
- Email security (Microsoft Defender for Office 365)
- Endpoint security (Intune, ASR rules, security baselines)
- Incident response using Microsoft Defender XDR

## Purpose
The goal of this repository is to document real-world security engineering concepts in a clear, practical way.  
All content is anonymised and based on lab environments or generalised scenarios.

## Topics Covered
- Conditional Access design considerations
- Privileged Identity Management (PIM)
- Phishing investigation and response workflows
- Endpoint hardening with Intune
- Defender Secure Score improvements

> This repository is for learning, sharing knowledge, and demonstrating practical security engineering skills.
